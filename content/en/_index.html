{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} Ouroboros is a peer-to-peer packet network prototype. It unifies all packet communications --whether it is two programs the same machine or two programs in computers on different continents -- using a single abstraction, which we call a flow. The Ouroboros architecture improves security, privacy and efficiency through simplicity. It provides a very compact API for writing distributed software and networked application libraries, with support for both unicast multicast communications. Being rebuilt from the ground up, Ouroboros is not directly compatible with IP or UNIX sockets, but it is interoperable with UDP/IP and Ethernet. {{% /blocks/lead %}}
