Join the Ouroboros community

Ouroboros is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Ouroboros? Find out more here:

  • Twitter: Follow us on Twitter to get the latest news!

Develop and Contribute

If you want to get more involved by contributing to Ouroboros, join us here:

  • Repository: Browse the repository
  • GitHub mirror: Mirror of the repository. May not be fully up-to-date.
  • Bitbucket mirror: Mirror of the repository. May not be fully up-to-date.
  • Slack: Join our slack workspace
  • Bugzilla: Send a bug report
  • Developer mailing list: Discuss development issues and send patches

You can find out how to contribute to these docs in our Contribution Guidelines.
